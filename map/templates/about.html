About

LaForge : Predictive Maintenance Software 1.0 (2017)

1.Purpose

Purpose of this software is to help maintenance team and manager to monitor their performance and to have a global overview of their machines. When enough data is saved in database, the software will attempt to make a prediction of which machine could experience a breakdown in a near future and share this information with the team in order to anticipate any severe production stop.

d. Prediction

A model is being created from the data input during the use of the software. For each spare parts the model will try to predict the life duration left before any problem occurs. Prediction is made in days and is here to help you to avoid production issues. You should maintain spare part before their date of expiration. If a spare part broke before the date predicted then you should input maintenance data and the model will be automaticaly updated.
